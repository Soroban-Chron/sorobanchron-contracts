# Contributing to sorobanchron-contracts

Thank you for contributing to `sorobanchron-contracts`.

## How to contribute

1. Fork the repository.
2. Create a feature branch from `main`.
3. Add tests for your contract logic changes.
4. Run `cargo test` locally before submitting a pull request.
5. Open a PR with a clear description and link to any related issue.

## Code style

- Keep Rust code compatible with `cargo fmt`.
- Use clear commit messages and small, incremental changes.
